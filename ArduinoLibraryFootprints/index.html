Project repo: ArduinoLibraryFootprintAnalyzer Github Repo

What you can see here:

  • Library name (either as defined in the ArduinoIDE Library JSON or in the library properties file)
  • Repository of the library
  • Tested version
  • Smallest Program storage usage in bytes*
  • Smallest Dynamic storage usage in bytes*
  • Platform argument as supplied to the compiler

*For Program Storage (PS) as well as Dynamic Storage (DS) 2 values are listed. The smaller is the additional storage used by the example when compared to an empty sketch
